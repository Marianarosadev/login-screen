# Vue login screen 

Esse projeto foi desenvolvido para simular o login de usuario em Vue.js

## 💻 Stacks

* **Vue.js**
* **HTML**
* **CSS/SCSS**
* **JavaScript**
* **tailwindcss**

## Project setup
``
npm install
``
### Compiles and hot-reloads for development
``
npm run serve
``
